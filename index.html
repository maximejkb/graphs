Graph Visualizer

This is a simple tool for visualizing graphs. I made it when I was learning about graph theory for the first time because I found traversal and search algorithms super interesting. I updated it later to include the Erdős–Rényi model.

Usage

Generate a graph using Erdős–Rényi by setting N and p, then clicking the New Graph button. p will by default be set to the connectedness threshold value of ln(N) / N. Since this is a sharp threshold, it's interesting to observe the connectedness of a graph with sufficiently large N at the threshold plus or minus a small epsilon.

Visualize an algorithm by checking the box to the left of the algorithm's name and clicking on a start node in the graph. For A*, click on a start node first and a target node second.

Use the animation speed slider to control the rate at which search // traversal algorithms propagate through the graph. Check Hide vertex labels and // or hide edge weights for a cleaner view.
